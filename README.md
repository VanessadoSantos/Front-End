## 🎯 Visão Geral e Objetivo do Negócio
O **Menina Sapeka Sexshop** é uma plataforma de comércio eletrônico desenvolvida para atender às demandas modernas de privacidade, velocidade e usabilidade. O projeto resolve problemas comuns desse nicho, como:
- **Discrição Absoluta:** Otimização de fluxos para garantir faturamento discreto.
- **Velocidade de Carregamento:** Otimização de imagens e Server-Side Rendering (SSR) para evitar o abandono de carrinho.
- **Navegação Intuitiva:** Categorização clara de produtos para facilitar a experiência do usuário.

## 🛠️ Stack Tecnológica e Decisões de Arquitetura
- **Next.js 15 (App Router):** Escolhido para renderização híbrida rápida (SSR/SSG), fundamental para o SEO de e-commerce.
- **TypeScript:** Tipagem estrita de dados de produtos, carrinho e checkout, reduzindo bugs em produção.
- **Tailwind CSS:** Estilização baseada em utilitários para criar um design moderno, limpo e responsivo.
- **Zustand:** Gerenciamento de estado global leve para o carrinho de compras.
- **Acessibilidade (WCAG):** Garantia de navegação por teclado e semântica HTML correta.

## 📦 Funcionalidades Principais
- **Catálogo de Produtos:** Listagem dinâmica com filtros por categoria, preço e tags.
- **Carrinho de Compras Persistente:** O cliente pode adicionar e remover itens e a quantidade é mantida no LocalStorage.
- **Checkout Simulado/Integrado:** Fluxo de compra fluido, focado na segurança de dados do cliente.
- **Modo Privativo / Discreto:** Tema visual e descrições otimizadas para discrição.
